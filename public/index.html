Developer API Reference

Integrasikan Gateway API ini dengan sistem eksternal (seperti Odoo, ERP, script Python, atau CRM) untuk mengirim pesan otomatis.

POST /api/send-message

Universal send message route (menentukan sesi di request body).

{
  "session": "waschen_utama", // Pilihan: "alora", "cleanox", "waschen_utama", "waschen_legenda", "waschen_citra", "waschen_canadian", "waschen_sentra_eropa", "waschen_raffles"
  "to": "628123456789",
  "message": "Halo, ini adalah pesan otomatis dari Gateway Waschen Utama!"
}
POST /api/sessions/:id/send

Direct send message route untuk sesi tertentu via URL path.

// Contoh URL: http://localhost:3000/api/sessions/waschen_utama/send
{
  "to": "628123456789",
  "message": "Halo, ini pesan langsung via endpoint Waschen Utama!"
}
GET /api/sessions

Mendapatkan status koneksi untuk seluruh sesi yang aktif.
