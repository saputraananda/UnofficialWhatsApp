Developer API Reference

Integrate this Gateway API with external systems (such as Odoo, ERP, Python scripts, or CRMs) to trigger automated messaging.

POST /api/send-message

Universal send message route.

{
  "session": "alora", // "alora" or "cleanox"
  "to": "628123456789",
  "message": "Hello, this is an automated message."
}
POST /api/sessions/:id/send

Direct send message route for a specific session.

// URL: http://localhost:3000/api/sessions/cleanox/send
{
  "to": "628123456789",
  "message": "Hello from Cleanox Business!"
}
GET /api/sessions

Retrieve connection state for all sessions.
